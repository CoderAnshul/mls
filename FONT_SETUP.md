# Narin Font Installation

The website is configured to use the **Narin** font family. To complete the setup, please copy your Narin font files to the `public/fonts` directory.

## Required Font Files

Please copy these files from your system to `d:/nexprism/aab-web/public/fonts/`:

1. `subset-Narin-Regular.woff2`
2. `subset-Narin-Regular.woff`
3. `subset-Narin-Medium.woff2`
4. `subset-Narin-Medium.woff`
5. `subset-Narin-Bold.woff2`

## Font Weights Configured

- **Regular** (400): subset-Narin-Regular
- **Medium** (500): subset-Narin-Medium  
- **Bold** (700): subset-Narin-Bold

## Fallback Fonts

If the Narin fonts are not found, the website will fall back to:
- Inter (from Google Fonts)
- System fonts

## Verification

After copying the fonts, refresh your browser at http://localhost:5173/ to see the Narin font in action!
